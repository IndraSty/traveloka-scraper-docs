# SignUp User

Create User Data

### Endpoint

```md filename="Base URL"
http://localhost:5000
```

```md filename="Method and Endpoint"
POST /api/auth/signup
```

### Request Body
```json filename="Request Body"
{
    "first_name": "firsName",
    "last_name": "lastName",
    "email": "example@gmail.com",
    "password": "YourPassword_123",
}
```

### Responses
You will get responses like this:

#### 201 (created)

```json filename="Response Body"
{
    "message": "User Created Successfully"
}
```

#### 400 (Bad Request - Validate Error)

```json filename="Response Body"
{
    "message": "Validate-Error"
}
```

#### 400 (Bad Request - Email Exist)

```json filename="Response Body"
{
    "message": "User with given email already exist!"
}
```

#### 500 (server error)

```json filename="Response Body"
{
  "message": "Internal server error"
}
```
