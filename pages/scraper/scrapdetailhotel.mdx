# Scrap Detail Hotel

Scraping Detail Hotel by Url

### Endpoint

```md filename="Base URL"
http://localhost:5000
```

```md filename="Method and Endpoint"
POST /api/users/current/scraphotel
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Request Body

```json filename="Request Body"
{
  "url": "https://traveloka.com/full-url-of-the-page"
}
```
